## 说明

全网首个专属于 `UniApp` 的编译工具插件，全面支持各平台、各环境、各语言、各类型文件、各编译工具。

## 功能

-   [路由模板](https://docs.965.ink/uniapp-router-view-loader/md/introduce.html#路由模板)
